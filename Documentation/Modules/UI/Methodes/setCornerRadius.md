# setCornerRadius

`setCornerRadius` is a function that changes the corner radius of a [**UIElement**](../UIElements.md).



***

###### Arguments

<table border="1">
  <thead>
    <tr>
      <th>Argument Name</th>
      <th>Type</th>
      <th>Description</th>
      <th>Notice</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td><code>radius</code></td>
      <td><code>double</code></td>
      <td>The new corner radius of a UIElement</td>
      <td>None</td>
    </tr>
  </tbody>
</table>



###### Example

```js
include("ui");

let box = ui.spawnBox();
box.setCornerRadius(15.2);
```
